## 📊 Dataset Availability (Hugging Face)

We have curated and released a massive, comprehensive benchmark dataset to evaluate synthetic handwriting detection models. It contains over **110,000 word-level images** across 13 state-of-the-art (SOTA) generation/synthesis engines (including both GAN-based, Transformer-based and Diffusion-based architectures).

The dataset is fully open-sourced and hosted on Hugging Face:

👉 **[ Hugging Face Dataset: Synthetic-Handwriting-Dataset ](https://huggingface.co/datasets/AgainsTurb/Synthetic-Handwriting-Dataset)**


> 📢 **Notice:** The core implementation of the framework is fully functional. The remaining scripts, extended configurations, and pre-trained checkpoints will be updated subsequently.
